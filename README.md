## 🚀 Overview

**MeDoType** strips away the clutter of traditional typing tests to deliver a laser-focused, high-performance typing experience. While the interface remains beautifully simplistic and distraction-free, underneath lies a robust web app powered by a real-time database ecosystem. Whether you are looking to casually practice your words-per-minute (WPM) or compete for the top spot on the global leaderboard, MeDoType has a mode built for you.

---

## ✨ Features

### Gameplay & Competition
* **Multiple Playstyles:** Jump into **Singleplayer** to test your limits or use **Practice Mode** to warm up your fingers risk-free.
* **Ranked Matchmaking:** Put your speed to the test in high-stakes competitive matches to build your ELO rating.
* **Dynamic Game Modes:** Fully customizable JSON-driven typing configurations, including:
    * *Timed Challenges* (15s, 60s)
    * *Word Count Sprints* (15 words, 30 words)
    * *Blitz & Sudden Death* modes for extreme accuracy challenges.
* **Live Leaderboards:** An automated, paginated global leaderboard tracking top speeds and competitive rankings instantly.

### Profile & Customization
* **Secure Authentication:** Complete account registration, login, and session persistence powered securely by **Supabase**.
* **In-Game Economy:** Earn custom platform currency just by typing and performing well.
* **The Cosmetics Shop:** Spend your hard-earned currency to unlock unique custom **Avatars** and premium interface **Themes** to personalize your dashboard.
* **User Discovery:** Comprehensive user searching tool to inspect profiles, history, and stats of friends or rivals.

### Under the Hood
* **Secure Admin Panel:** A gated backend dashboard built for administrators to easily moderate users, manage data, and maintain platform health.
* **Anti-Cheat Protection:** Integrated verification mechanics to validate typing telemetry, filtering out automated scripts and keeping the competitive ladder fair.

---

## 🛠️ Tech Stack

MeDoType is engineered utilizing a modern, lightweight tech stack focused on raw rendering speed and secure serverless architecture:

* **Frontend:** Semantic HTML5, Vanilla JavaScript (ES6+ Async/Await API architecture).
* **Styling:** Modern CSS3 utilizing strict CSS Variables (`:root`) for instant, seamless theme-switching.
* **Database & Auth:** [Supabase](https://supabase.com/) (PostgreSQL backend tracking user authentication, user currency, profile analytics, and leaderboard states).
* **Icons:** FontAwesome v6.4 library.
